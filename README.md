# KgkWmsWebRe

一个基于目录分层初始化的 `React + TypeScript` 项目，使用 `Vite` 作为开发和构建工具。

## 目录

- `app` 应用入口与全局配置
- `pages` 页面层
- `layouts` 布局层
- `modules` 业务模块层
- `shared` 通用组件、工具、类型、请求封装
- `assets` 静态资源
- `tests` 测试目录

## 启动

```bash
npm install
npm run dev
```

## 其他命令

```bash
npm run build
npm run preview
npm run typecheck
```
